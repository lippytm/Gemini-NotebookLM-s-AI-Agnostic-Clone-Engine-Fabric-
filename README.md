# Gemini NotebookLM AI-Agnostic Clone Engine Fabric

This repository is the integration hub for Charles Lipshay's lippytm.AI knowledge, agent, and clone infrastructure.

## Integrated systems

- Gemini AI and NotebookLM notebooks as source-grounded research and knowledge workspaces
- AI-Agnostic Clone Engine Fabric as the portable identity, memory, prompt, policy, and workflow layer
- Hermes as the routing, dispatch, messaging, and repository coordination layer
- Fable 5 as a narrative, simulation, content-generation, and multimodal production lane
- GitHub as the canonical version-control, documentation, governance, issue-tracking, and deployment backbone
- Prompt #11 as the operating prompt and product-manufacturing framework

## Design rule

No individual AI provider is the permanent system of record. Knowledge, prompts, manifests, policies, provenance, and reusable outputs should be exported into open, versioned formats stored in GitHub-compatible repositories.

## Core folders

- `docs/` architecture, governance, runbooks, and operating procedures
- `manifests/` notebook, agent, clone, model, and repository manifests
- `prompts/` provider-neutral Prompt #11 and task templates
- `knowledge/` normalized source indexes and citation metadata
- `connectors/` Gemini, NotebookLM, Hermes, Fable 5, GitHub, and future provider adapters
- `schemas/` JSON Schema and data contracts
- `workflows/` orchestration recipes and GitHub Actions
- `riskgate/` safety, copyright, privacy, security, and evidence controls

See `docs/INTEGRATION_ARCHITECTURE.md` for the initial merge blueprint.
